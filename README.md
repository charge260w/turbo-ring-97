# Turbo Ring 97

一个街机风格车辆格斗小游戏。角色是 Nissan GT-R、Mercedes S、Porsche 911、Toyota Supra、Corvette ZR1、Audi R8 等车款灵感的原创赛车战士。

## Play

Open `index.html` directly, or serve the folder with any static web server.

Player 1:

- `A` / `D`: move
- `W`: hop
- `S`: guard
- `J`: ram
- `K`: drift dash
- `L`: special

Player 2:

- Arrow keys: move / hop / guard
- `1` / `2` / `3`: ram / drift dash / special

Mobile:

- Tap `全屏`, choose a car, then use the on-screen controls.
